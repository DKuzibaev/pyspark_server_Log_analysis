# Web Log Analysis Project

This repository contains two main files:

1. **main.py**  
   Script for generating web server data (logs for analysis).  
   !This script is only provided to demonstrate how `web_log_analysis.ipynb` works. You can replace it with your own log file.

2. **web_log_analysis.ipynb**  
   Jupyter Notebook with web log analysis:
   - Top 10 most active IPs by number of requests.
   - Number of requests per HTTP method.
   - Count of requests with 404 response code.
   - Total response size grouped by date.
